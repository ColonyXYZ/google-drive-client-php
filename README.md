# Google Drive Client PHP

The php library to Google Drive API operate.

## Installation

### Using Composer

composer.json
```
{
    "require": {
        "soramugi/google-drive-client-php": "*"
    }
}
```
    curl -s http://getcomposer.org/installer | php
    php composer.phar install
